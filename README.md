# tableau-sales-analysis
Tableau sales dashboard project
## Project Description
Designed an interactive Tableau dashboard to analyze sales and profit trends.  
Due to tool limitations, the final dashboard is shared as a PDF.

## Tools Used
- Tableau Desktop
- MS Excel

## Dashboard Highlights
- Sales and profit analysis by region
- Category-wise performance
- KPI summary and trends

## Files Included
- Sales Dashboard (PDF)
- Sales ScreenShots
- Sales_Data.xlsx (dataset used for analysis)
